# University LMS UI

A responsive, professional Learning Management System (LMS) user interface developed using **HTML5** and **CSS3** only. This project simulates a real-world university platform for students and faculty.

## ðŸš€ Features

*   **Responsive Design:** Fully optimized for Desktop, Tablet, and Mobile devices.
*   **Modern UI:** Clean, professional aesthetic using CSS Grid and Flexbox.
*   **Six Core Pages:**
    1.  **Login Page:** Clean authentication interface.
    2.  **Student Dashboard:** Overview of courses, attendance, and grades.
    3.  **Course List:** Grid view of enrolled courses with status indicators.
    4.  **Course Detail:** Two-column layout with syllabus and video player placeholder.
    5.  **Assignments:** Table view for managing tasks with file upload UI.
    6.  **Profile:** User profile management with edit capabilities.
*   **Zero Dependencies:** No frameworks (Bootstrap/Tailwind) or JavaScript used.
